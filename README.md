# Sweet Delights | Artisanal Pastry Shop

A beautiful, modern web application for an artisanal pastry shop and bakery. Built with **Astro 5** and **Tailwind CSS 4**, featuring a premium design aesthetic with smooth transitions and a responsive layout.

![Sweet Delights Hero](https://images.unsplash.com/photo-1555507036-ab1f4038808a?q=80&w=2000)

## ? Features

- ?? **Artisanal Showcase**: Beautifully designed sections for cakes, pastries, and cookies.
- ?? **Responsive Design**: Optimized for all devices, from mobile to desktop.
- ?? **Premium Aesthetics**: Carefully curated color palette (#5c4033, #d4a373, #faedcd) and typography (**Playfair Display** & **Inter**).
- ?? **Contact System**: Ready-to-use contact form and shop information section.
- ?? **Performance First**: Built with Astro for lightning-fast loading speeds.
