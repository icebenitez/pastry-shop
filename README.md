# Sweet Delights | Artisanal Pastry Shop

A beautiful, modern web application for an artisanal pastry shop and bakery. Built with Astro 5 and Tailwind CSS 4, featuring a premium design aesthetic with smooth transitions and a responsive layout.

![Sweet Delights Hero](https://images.unsplash.com/photo-1555507036-ab1f4038808a?q=80&w=2000)

## Features

- Artisanal Showcase: Beautifully designed sections for cakes, pastries, and cookies.
- Interactive Menu: Full digital menu powered by Astro Content Collections with interactive hover previews.
- Our Story: Dedicated about page detailing the heritage and craft of Sweet Delights.
- Responsive Design: Optimized for all devices, from mobile to desktop.
- Premium Aesthetics: Carefully curated color palette (#5c4033, #d4a373, #faedcd) and magazine-style typography (Playfair Display & Inter).
- Contact System: Ready-to-use contact form and shop information section.
- Performance First: Built with Astro 5 for lightning-fast loading speeds and smooth view transitions.

## Tech Stack

- Framework: Astro 5
- Styling: Tailwind CSS 4
- Content: Astro Content Collections (Markdown-based menu)
- Fonts: Google Fonts (Playfair Display, Inter)
- Animations: Native Intersection Observer and CSS transitions for premium feel.

## Getting Started

### 1. Project Structure

/
|-- public/          # Static assets
|-- src/
|   |-- assets/      # Image and media assets
|   |-- components/  # Reusable Astro components
|   |-- content/     # Menu items (Markdown)
|   |-- layouts/     # Page layouts
|   |-- pages/       # Project routes (index, about, menu, 404)
|   |-- styles/      # Global CSS and Tailwind configuration
|-- package.json

### 2. Commands

All commands are run from the root of the project:

| Command | Action |
| :--- | :--- |
| npm install | Installs dependencies |
| npm run dev | Starts local dev server at localhost:4321 |
| npm run build | Build your production site to ./dist/ |
| npm run preview | Preview your build locally |
